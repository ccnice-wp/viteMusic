# musicWebsite
基于Vue.js和Spring Boot前后端分离的音乐网站项目
## 主要功能
### 前台
* 注册登录
* 播放音乐
* 检索音乐
* 用户自建歌单
### 后台
* 管理用户
* 添加歌手
* 上传音乐
* 添加歌单
